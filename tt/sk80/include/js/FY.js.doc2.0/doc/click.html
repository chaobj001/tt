$(document).click()函数

$(document).click().函数是FY.JS的绑定函数。

此函数用于给选择ID绑定函数在单击时触发目前不支持Class，今后也不会支持。

此函数适用于XHTML跟JS的逻辑分离。

$(document).click()函数 - 使用方式


$(document).ready(function(){
/*
如果要把js代码写在head区域那么必须包裹在ready函数里面.
否则click将不会执行.
*/
$('test').click(function(){
alert('风吟猪');
//这里可以写函数以及代码
})
})

<a href="###" id='test'>点我弹出风吟猪</a>

那么，点击 id 为 test 的链接就可以为 test添加事件了.这是onclick的替代函数.

本文当ajax提交的例子也是使用click函数的。
