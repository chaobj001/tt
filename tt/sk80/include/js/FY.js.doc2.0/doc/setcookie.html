$().setcookie()

$().setcookie() 是对cookie操作的函数，写入一个cookie. 本函数没有返回值

代码如下。

$(document).setcookie({
    name: "fengyin", // cookie名字
    value: "13456",  //cookie 值
    expires: "365",  //生存天数 删除一个cookie只要把日期设置为-1天
    path: "/",      //有效路径
    domain: "127.0.0.1",  // 指定域名
    secure: false   //是否加密
});
