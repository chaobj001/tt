$(document).ajax()

FY.JS库的ajax操作函数.该函数一共需要设置三个参数 url,data,success 如果执行成功则返回一个参数responseText.

FY.JS库强制对ajax函数使用UTF-8编码跟POST传输数据，因此在服务端请用$_POST方式获取数据，不支持$_GET，由于此库是迷你库，因此不加入诸多的功能。比如 超时啊，数据类型啊。统一用responseText数据类型、

函数使用方式如下：

$(document).ajax({
    url: 'reg.php',//POST的路径
    data: "user=" + $('user').val() + '&pwd='+$('password').val(), //参数
    success: function(responseText) {
        $("tips").html(responseText) //传入函数的值以便给你操作。
    }
});
