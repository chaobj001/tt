FY.JS说明

有一些时候，为了使用Jquery的ajax却要拖上它50KB的一个大文件，对于网站来说是得不偿失的,风吟曾经尝试过精简一个Jquery库，让他减小体积。但是后来实在被它的结构弄得头晕眼花，三天不能恢复我的脑力。得得得…… 我惹你不起我还写你不起么。这么个花哨产物，50KB比我网页文件还大，我用不起你。我自己写一个！

于是FY.JS就这样蛋生了，一个超级小的迷你JavaScirpt库。只封装了简单的DOM和Ajax以及Cookie操作。这样的一个小库对于一般小程序来说已经够用了。也许风吟并不需要Jquery的什么动画啊。扩展啊，插件啊，风吟只是要Jquery语法的方便以及小巧实用几个功能而已。本库已经通过 各大流行浏览器的支持，用的都是基本的语法可以保证以后兼容性。此库缺点也显而易见。没有遍历DOM，我写这个库的初衷已经很明白，封装一个简单使用类似于Jquery语法的库。功能只有常用的，不会有花哨.关键是实用方便。

简单就是一种美...
