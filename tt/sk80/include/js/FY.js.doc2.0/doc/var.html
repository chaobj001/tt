$()函数

$().函数是FY.JS最为核心的功能函数。

此函数用于选择 ID 目前不支持Class，今后也不会支持。那么小的一个库无法遍历全部DOM树。事实上ID已经够用。

由于不类似Jquery用“#”识别ID，用“.”选择Class因此也不需要 $('#test'); 只需要 $('test');

$()函数 - 使用方式

$('选择的ID').XXX();

XXX为要操作的函数。支持的函数请看左侧列表。

$()函数 - 特别说明

在操作DOM时候必须包含在$().ready()函数中，具体请看$().ready().

除了用 $('test') 作为ID选择器之外还可以用 _FY.('test') 防止冲突. _FY() 等效于 $()
